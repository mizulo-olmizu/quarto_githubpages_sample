自動車市場の燃費分析

1 分析の背景と目的

1970〜80年代の自動車市場では、オイルショックを背景に燃費性能が重要な競争要因となりました。 本分析では、米国・欧州・日本の3地域の自動車データを用いて、以下の問いに答えます。

  1. 地域間で燃費にどの程度の差があるか？
  2. 燃費を決定づける要因は何か？
  3. 年代とともに燃費はどう変化したか？
Code 
import altair as alt
import polars as pl
from great_tables import GT
from vega_datasets import data as vdata

cars = (
    pl.read_json(vdata.cars.filepath)
    .with_columns(pl.col("Year").str.slice(0, 4).cast(pl.Int32).alias("Year"))
    .drop_nulls()
)

2 1. 地域別の燃費概況

まず、地域ごとの基本統計量を確認します。

Code 
summary = (
    cars.group_by("Origin")
    .agg(
        pl.col("Miles_per_Gallon").count().alias("台数"),
        pl.col("Miles_per_Gallon").mean().alias("平均燃費"),
        pl.col("Miles_per_Gallon").median().alias("中央値"),
        pl.col("Miles_per_Gallon").min().alias("最小"),
        pl.col("Miles_per_Gallon").max().alias("最大"),
        pl.col("Horsepower").mean().alias("平均馬力"),
        (pl.col("Weight_in_lbs").cast(pl.Float64) * 0.4536).mean().alias("平均重量kg"),
    )
    .sort("Origin")
    .rename({"Origin": "地域"})
)

(
    GT(summary)
    .tab_header(title="地域別 自動車スペック概要")
    .fmt_number(columns=["平均燃費", "中央値", "平均馬力", "平均重量kg"], decimals=1)
    .data_color(columns="平均燃費", palette=["#fee0d2", "#de2d26"])
)

4.2 年代別サマリ（ナノプロット付き）

各地域の燃費推移をスパークラインで一覧します。

Code 
yearly_avg = (
    cars.group_by(["Origin", "Year"])
    .agg(pl.col("Miles_per_Gallon").mean().round(1))
    .sort("Year")
)

spark_data = []
for origin in cars["Origin"].unique().sort().to_list():
    vals = yearly_avg.filter(pl.col("Origin") == origin)["Miles_per_Gallon"].to_list()
    line = " ".join(str(v) for v in vals)
    g = cars.filter(pl.col("Origin") == origin)
    diff = round(
        g.filter(pl.col("Year") >= 1978)["Miles_per_Gallon"].mean()
        - g.filter(pl.col("Year") < 1975)["Miles_per_Gallon"].mean(),
        1,
    )
    spark_data.append({"地域": origin, "燃費推移": line, "後期-前期差": diff})

spark_df = pl.DataFrame(spark_data)

(
    GT(spark_df)
    .tab_header(
        title="地域別 燃費推移スパークライン", subtitle="1970〜1982年の平均燃費"
    )
    .fmt_nanoplot("燃費推移", plot_type="line")
    .fmt_number("後期-前期差", decimals=1)
    .data_color(columns="後期-前期差", palette=["#fee0d2", "#41ab5d"])
)

5 まとめ

  • 日本車は一貫して高い燃費性能を維持し、後期にはさらに改善
  • 米国車は重量・排気量が大きく燃費が低いが、後期に改善傾向
  • 欧州車は中間的な位置づけで、バランスの取れたスペック
  • 燃費に最も影響する要因は車両重量であり、次いで排気量・馬力が続く
